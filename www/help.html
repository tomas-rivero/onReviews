Esta aplicacion fue desarrollada por 2 chicos durante el ultimo año de la secundaria y posteriormente se agregaron nuevas funciones y mejoras en base a su conocimento y experiencia con distintos proyectos.
Fue creada por y para fines estudiantiles, sin fines de lucro. Se mantiene sujeta a cambios/actualizaciones/mejoras en un futuro.

Para mas informacion sobre ellos, entrar al apartado de "Sobre los creadores" (actualizada al año 2023).


Stack de tecnologias usadas:
  • Html
  • Css
  • Javascript
  • Framework7
  • Api Google Maps
  • Api TMDB (The Movie DataBase)

Ante cualquier inconveniente, error o "bug" con la aplicacion comuníquese con alguno de sus propietarios.
